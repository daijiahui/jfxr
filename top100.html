BigDataBench100

Overview

The BigData100 is an open source project for benchmarking and ranking big data systems. The benchmarks we use are selected from BigDataBench. Currently, the big data systems include Hadoop, Spark, Flink, Hive on Hadoop, Impala, and the workloads cover offline batch processing, iterative machine learning, and interactive query processing from different domains. There would be more systems and workloads in the future.
